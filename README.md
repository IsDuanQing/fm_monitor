# Qt Workspace

This repository is a personal Qt workspace that contains multiple practice and project directories.

## Highlighted Project

### InformationShow

`InformationShow` is a Qt Widgets desktop application for TCP-based monitoring and display of environmental sensor data.

Main features:

- TCP server/client mode switching
- Real-time display of sensor values
- Noise chart visualization with Qt Charts
- Debug window for message inspection
- Database window for historical data viewing

Project path:

- `InformationShow/`
